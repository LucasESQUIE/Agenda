# todo-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
1. npm run serve pour le frontend
2. aller dans todo-app-server et faire npm run dev pour lancer le backend
```

### Compiles and minifies for production
```
npm run build
```
